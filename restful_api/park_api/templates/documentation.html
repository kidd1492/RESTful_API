/parks

start
Type: Integer
Usage: Query parameter
Description: Get the next [limit] results starting with this number. Default is 0.

limit
Type: Integer
Usage: Query parameter
Description: Number of results to return per request. Default is 25.

Example: api/v1/parks?start=0&limit=2
